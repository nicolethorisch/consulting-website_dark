# Nicole Thorisch – KI & Analytics Consulting

Website für Nicole Thorisch | KI-Automatisierung für Immobilienverwaltungen, Banken, Versicherungen und Industrie.

**Repo:** [nicolethorisch/consulting-website_dark](https://github.com/nicolethorisch/consulting-website_dark)

## Tech Stack

- **Next.js 16** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **Framer Motion**

## Development

```bash
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy on Vercel

1. Verbinde das GitHub-Repository [consulting-website_dark](https://github.com/nicolethorisch/consulting-website_dark) mit Vercel
2. **Root Directory:** leer lassen (Projekt liegt am Repo-Root)
3. **Framework Preset:** Next.js (wird automatisch erkannt)
4. **Build Command:** `next build` (Standard)
5. **Output Directory:** leer lassen (Next.js Standard)
6. **Node.js Version:** 18 oder höher

Vercel erkennt die Konfiguration aus `vercel.json` und `package.json` automatisch.
